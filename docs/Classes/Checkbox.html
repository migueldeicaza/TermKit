Checkbox

public class Checkbox : View

The Checkbox View shows an on/off toggle that the user can set

Example:

 c = Checkbox ("Toggle me")
 c.x = Pos.at (0)
 c.y = Pos.at (0)
 c.width = Dim(30)
 c.height = Dim (1)
